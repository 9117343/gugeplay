王耀珉:金融风暴下中国职业装行业的形势和任务 < style=" TEXT-INDENT: 71.95pt; mso-char-indent-count: 4.48"> < style=" TEXT-INDENT: 21pt; mso-char-indent-count: 2.0">中国职业装企业家峰会是职业装行业每年一度的企业年会，也是行业例会，是总结当年规划来年的会议，因此，在行业内具有深远的影响。每年会议发出的声音都在互联网上广泛传播，影响甚远。 < style=" TEXT-INDENT: 21pt; mso-char-indent-count: 2.0">2008年是中国职业装发展十年的一个历史阶段，去年4月我们在天津举行了“中国职业装十年庆典”，在十年庆典上，我们总结了中国职业装十年发展的经验，评选出了中国职业装十大领军企业，举办了中国职业装发展战略研讨会和职业装十年回顾展，中国职业装在行业在发展中前行，走向新的辉煌。但是，由美国次贷危机引发的金融风暴正席卷全球，世界经济受到严重影响，金融动荡带来的不稳定因素还在增加，欧美国家正在采取措施救市，中国政府最近也出重拳，出台4万亿救市资金，以确保中国经济平稳较快增长。4万亿令世界震惊，也是中国历史上少见的宏观经济举措。因此，在这样的形势下，中国职业装行业面临什么问题，我们职业装行业应该有些什么作为，是我们这次会议的主要议题。 < style="MARGIN: 0cm 0cm 0pt 42pt; TEXT-INDENT: -21pt; mso-list: l1 level1 lfo2; tab-stops: list 42.0pt">一． 当前的行业环境和外部形势 < style="MARGIN: 0cm 0cm 0pt -0.2pt; TEXT-INDENT: 21pt; mso-char-indent-count: 2.0; mso-para-margin-left: -.02gd">在世界金融风暴袭来之日，中国服装工业受到了重创。外向型企业特别是广东一些服装企业订单减少，甚至很多企业倒闭。由此而引发的企业用工需求减少，导致失业工人增加，内地出现返乡潮。在职业装行业，经我们调查，也有一些企业面临上述问题，但总的情况还不很严重。原因是职业装企业绝大部分是内销，面向的是国内市场。在沿海一带的企业中，因为外贸形势严峻而影响生产的企业确实存在，一些企业倒闭，有的问题直接原因是没有定单，货物积压，也有的问题出在资本运作失误和资金链断裂上。从中国职业装行业目前整个的运行情况看，形势还是比较乐观的，甚至可以说是风景这边独好。 < style="MARGIN: 0cm 0cm 0pt -0.2pt; TEXT-INDENT: 21pt; mso-char-indent-count: 2.0; mso-para-margin-left: -.02gd">尽管如此，我们所面临的外部形势还是比较严峻的和不容忽视的。首先是金融危机的后果还没有真正显现，它的走势和带来的影响，现在还很难估计，至少两到三年才能度过，因此它的滞后作用，作为职业装行业绝对不能麻痹，要审时度势，严阵以待。 < style="MARGIN: 0cm 0cm 0pt 42pt; TEXT-INDENT: -21pt; mso-list: l1 level1 lfo2; tab-stops: list 42.0pt">二． 我们的任务 < style="MARGIN: 0cm 0cm 0pt 21pt">基于上述形势的分析，作为职业装行业要继续承担为社会服务，为经济建设服务的重任

必须强调我们职业装行业的价值观、服务理念和社会责任。要象我们在防非典战役中发挥行业作用那样，在度过这次百年不遇的金融风暴中，再立新功。 < style=" TEXT-INDENT: 21.75pt">根据中央最近做出的决策，职业装行业应该认真研究几个问题。 < style=" TEXT-INDENT: 21.75pt">一是在全球金融危机日趋严峻，我国经济遭受冲击日益显现的背景下，中央对宏观政策做出重大调整，正在实施积极的财政政策和适当宽松的货币政策； < style=" TEXT-INDENT: 21.75pt">二是年内中央紧急拿出1000亿人民币投入实体经济，出台进一步扩大内需的十条措施，以促进国民经济平稳较快增长，预计可带动4000亿人民币的社会总投资规模； < style=" TEXT-INDENT: 21.75pt">三是从今年第四季度到2010年底，中国用于这些领域的投资将增加4万亿人民币。在这里我要强调4万亿经济刺激计划这个概念，4万亿将拉动GDP增长1.8个百分点，从而使明年GDP增长率保持在9%以上，因为低于9%就可能出现就业率低，社会保障不利，消费不振的不景气景象。4万亿，这个数字将魔化出一个新兴的中国，开启新的30年经济增长期。因此，今天的会议就是给大家一个强烈的信号，做好充分的思想准备，在世界经济不景气的形势下，我们要抓住机遇，迎接挑战，在未来的10年、30年中创造中国职业装的新的辉煌。今年4月我们在十年庆典中规划了中国职业装未来十年的发展前景，结合今天这次会议，我们大家要集思广益，把眼光放在世界经济格局变化的视点上，谋求新的更大的发展。 < style="MARGIN: 0cm 0cm 0pt 45pt; TEXT-INDENT: -18pt; mso-list: l0 level1 lfo1; tab-stops: list 45.0pt">1． 抓住拉动内需机遇，深度开发职业装市场。中央的十项拉动内需的措施发出了扩大投资规模，加大基础设施建设，以投资带消费和合理扩大信贷规模的信号。这些措施的出台将拉动安居工程建设，道路交通建设等重大项目，同时各地也在制定重大项目的申报工作。根据当前宏观经济形势和铁路发展规划，2009年计划完成基建投资6000亿元，根据定额测算分析，完成这一规模的投资需要2000万吨钢材、1.2亿吨水泥，能够提供600万个就业岗位。加快推进大规模铁路建设对拉动投资和内需的作用十分明显。另外，盖1O的房子，可以解决16个人就业，120公斤钢材，300公斤水泥，0.65平方米的混凝土。因此，新一轮的经济发展必定带来大批工人的就业和工程项目的开工，带来对职业装的需求。在深度和广度上开发职业装市场，主动出击，上门服务，迎接招投标等,将是我们职业装企业获得发展的一个新的机遇。 < style="MARGIN: 0cm 0cm 0pt 45pt; TEXT-INDENT: -18pt; mso-list: l0 level1 lfo1; tab-stops: list 45.0pt">2． 充分享受增殖税转型政策，争得发展机会。“增值税转型”是国务院十项措施让利于企业的阳光政策，不仅对那些获得较大抵扣税额的企业有所利好，对职业装企业同样能够得到实惠。作为小额纳税人的职业装企业，在购置固定资产的投资活动中增值税的下降，不仅可以直接转化为利润，而且能够获得更多的积累，为企业的在发展预备充足的资金。因此，把握好机遇，在科技进步，引进先进设备上有所作为，也是我们在这一轮经济发展中应当很好把握的一个机会。 < style="MARGIN: 0cm 0cm 0pt 45pt; TEXT-INDENT: -18pt; mso-list: l0 level1 lfo1; tab-stops: list 45.0pt">3． 以信誉赢得担保。职业装企业95%以上是小企业，多年来，金融机构对小企业在信贷上存在偏见，因此在通过信贷渠道获得资金支持上存在难度。十项措施要求金融机构进一步加大对中小企业的信贷支持和贷款贴息的力度。因此，在此轮刺激经济增长的大环境中，我们职业装企业，一方面要积极争取担保公司和金融机构的支持，另方面也要在保持畅通的现金流的同时，加强自身管理和品牌建设，及时与金融机构沟通，赢得信誉，从而获得资金支持。 < style="MARGIN: 0cm 0cm 0pt 45pt; TEXT-INDENT: -18pt; mso-list: l0 level1 lfo1; tab-stops: list 45.0pt">4． 积极履行社会责任。最近两年，职业装企业在履行社会责任等方面遇到了许多问题，职工保险问题，员工休假问题，涨工资问题，提高待遇问题，改善住宿环境问题等等，加大了企业成本，这些是随着国家法律法规不断完善和国际舆论的压力下，职业装企业绕不过的问题。因此，在新一轮经济增长期中，中国职业装企业要获得发展，要规范，必须要积极创造条件，履行社会责任，在4万亿资金支持经济，十项举措拉动内需的大环境下，积极进取，有所进步，获得发展，迎接挑战。 < style="MARGIN: 0cm 0cm 0pt 27pt"> < style="MARGIN: 0cm 0cm 0pt 27pt">
