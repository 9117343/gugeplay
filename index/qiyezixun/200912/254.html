-世城-职业装情景剧表演---对职业装-秀-的变革与创新

2009-05-21 07:57　来源：工作服制服　作者：唐山高档男装 已有 人关注

“世城”职业装情景剧表演 < style=" LINE-HEIGHT: 26pt; TEXT-ALIGN: center; mso-line-height-rule: exactly" align=center> -----对职业装“秀”的变革与创新 < style=" LINE-HEIGHT: 26pt; mso-line-height-rule: exactly"> < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">过去，大凡时装表演，或是职业装表演，都离不开T台、模特、灯光、音响……等等。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">后来，随着社会的变进，这种表演的时尚元素越来越多，商业气氛越来越重，便有人称它为“秀”了。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">不论是表演，还是“秀”，它们的基本格式是：几个美女或帅男，穿上几套衣服，踱着猫步，在T台上走来走去。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">如今，这一模式已经悄然发生着改变。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">近日，记者在一场题为“凝聚每份爱”的大型职业装情景剧表演会上，身切地感受到了这一改变：好一场别开生面的职业装情景剧表演！ < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">表演是由我国职业装行业久负盛名的企业，并获得中国职业装十大领衔品牌的武汉世城职业服饰公司为庆祝公司成立十五周年而精心策划、编排、组织的，乃庆典活动主题。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">出席庆典活动，观看职业装情景剧表演的有当地省、市政府有关领导，省市公安、司法、法院、检察、工商、税务、城管等部门负责人，所在街区党政负责人，武钢集团、神龙汽车、东风本田、湖南“三一”重工、江西七一三厂、河南漯河等六市警服调剂中心、北京铁路局、兰州铁路局等客户代表各方嘉宾600多人。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">该情景剧表演分“扬帆启航”、“播撒美丽”、“弄潮物语”、“绿色使命”、“青春放歌”、“和谐时代”六大篇章，几乎囊括了职业装的所有门类，如服务行业装、工装、制式服装、特种防护服装、校服和休闲职业装等。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">之所以说它是对职业装“秀”的变革与创新，在于它一改传统的“走秀”为“演秀”，把单调的服装表演，与劳动情景、工作情景、生活情景有机结合起来，配上相应的表演道具，背景动画形象衬托和旁白对话解说，很好地诠释了世城多年一贯奉行的理念――“工作着是美丽的”，多角度地传递了职业装时尚化、功能化、环保化、休闲化的发展趋势。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">除此之外，情景剧表演还从剧场式的T台搬到了圆形的歌舞厅；演员中除模特外还增加了专业演员和群众演员；情景剧表演过程中还穿插了歌舞、杂技、小品、朗诵等文艺形式。整个表演高潮迭起，令人耳目一新，具有较高的艺术品位和较强的视觉冲击力及文化感染力。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">比如“播撒美丽”中，汽车工人身着亮丽的功能化工装，手拿焊枪、或扳手、或锤子等，围绕汽车模型展现各种工作状态，充分揭示了职业工装形象表达与安防功能结合的深刻内涵。又如“弄潮物语”中，城管执法人员身着威严而亲和的制服，穿梭在城市的大街小巷，面对各种违章和“摊点”，他们动之以情，晓之以理的执法举措……，既给人美好的视觉享受，又感受到了社会的和谐。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">世城职业装情景剧表演虽然只有短短的60分钟，但它对职业装“秀”是一种变革与创新，是一次有益而生动的探索和尝试，从中我们不难看到，武汉世城职业服饰公司之所以能够在十五年的发展历程中取得骄人的业绩，头上的荣誉光环一项又一项，与他们的坚韧、创新和持续求变的态度和做法不无关系。 < style=" TEXT-INDENT: 28.5pt; LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">哦，弄潮者，往往比别人更出新，更出彩，更受青睐！ < style=" LINE-HEIGHT: 26pt; mso-line-height-rule: exactly">来源：武汉世城职业服饰公司 郝建国
