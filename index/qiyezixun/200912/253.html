长春际华三五零四职业装有限公司

2009-08-11 03:48　来源：唐山商务男装　作者：职业装加工厂 已有 人关注

长春际华三五零四职业装有限公司始建于1951年，位于吉林省长春市经济技术开发区东南湖大路2351号，是东北地区唯一以生产军用服装和行业服装为主的综合性企业，隶属于际华集团有限公司。1986年以来先后被总后勤部、吉林省、长春市评为先进企业，国家二级企业，是ISO9001：2000质量认证达标企业。1989年以来，有14种产品荣获国优、军优、部优，其中八七校官夏常服获得国家银质奖。公司生产的军用产品连续14年在全军质量评比中均列前三名。04士兵冬服在全军质量统检中获第一名。先后被国家工商总局、吉林省工商局、长春市工商局评为“守合同 重信用”企业，吉林省诚信企业示范单位，长春经济技术开发区50强企业。 < style=" TEXT-INDENT: 28pt; LINE-HEIGHT: 17pt; mso-char-indent-count: 2.0; mso-char-indent-size: 14.0pt; mso-line-height-rule: exactly">长春际华三五零四职业装有限公司是有着55年悠久历史的老军工企业，一直以生产部队制式服装和行业服装为主，无论从企业规模和生产组织形式等各个方面都特别适合行业服装的生产，公司技术力量雄厚，具有先进的生产设备、完善的管理体制和完备的质量检测手段，设有专业的新产品开发机构，能自行设计、生产各种军、民用制式服装。公司拥有各类高、中级工程技术人员67人，工厂有先进的缝纫设备1522台，其中从美国、日本引进的特种先进机器1112台。公司1992年、2000年分别引进两套美国格柏公司CAD自动排版辅助设计系统，并于2003年工厂引进了美国格柏公司CAM电脑自动裁床系统。目前工公司的设备已达到世界先进水平. < style=" TEXT-INDENT: 28pt; LINE-HEIGHT: 17pt; mso-char-indent-count: 2.0; mso-char-indent-size: 14.0pt; mso-line-height-rule: exactly">多年来我公司一直承担省、市有关领导的服装、沈阳军区及东北三省武警消防、边防部队所有特体服装和总政歌舞团演员的演出服的制作，有着丰富的行业服装制作经验和一大批优秀的人才。公司能够在最短的时间内以最快的速度、最优质的产品、最满意的服务达到客户的需要。 自改革开放以来，公司在把军用服装作为企业主要产品的同时，积极面向国内外市场，根据客户的需求设计和生产了公安、工商、税务、法院、检察院、油田、港口、邮政、铁路等行业的制式服装。1991年以来开展对外贸易，先后为美国、日本、南韩、意大利等国加工出口西服、夹克衫、风雪衫、运动服等产品。并于1998年获国家外经贸部颁发的自营企业进出口证书。近两年来，又自营出口瑞士、韩国、荷兰的工装和运动服，直接创汇数百万美元。工厂信守合同，重视售后服务，产品在国内外市场赢得了用户广泛的赞誉。 < style=" LINE-HEIGHT: 14pt; mso-line-height-rule: exactly"> < style=" LINE-HEIGHT: 14pt; mso-line-height-rule: exactly"> < style=" LINE-HEIGHT: 18pt; mso-line-height-rule: exactly">地址：吉林省长春市经济技术开发区东南湖大路2351号 邮编：130033 < style=" LINE-HEIGHT: 18pt; mso-line-height-rule: exactly">电话：0431―84602329 传真：84602331 < style=" LINE-HEIGHT: 18pt; mso-line-height-rule: exactly">网址：www.cn3504.com 邮箱：cc3504@126.com
